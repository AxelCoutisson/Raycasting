Projet Java 
RayCasting 


L'objectif de ce projet est de pouvoir se déplacer sur une carte en 3D grâce au Raycasting,
ainsi que de pouvoir éditer, sauvegarder et importer une carte pour se déplacer dessus.
C'est donc un petit moteur de jeu Raycasting.

Lorsque vous lancerez l'application, vous arriverez sur le menu avec 4 choix possible :
->Load: charger une carte pour se déplacer dessus.
-> Edit: éditer une carte et la sauvegarder.
-> Settings: changer de résolution.
-> Quit: quitter l'application.

Lorsque vous souhaitez charger une carte, vous devez choisir une carte au format ".txt",
ou bien prendre la carte par défaut qui vous permettra d'avoir une première visualisation du projet.

Lorsque vous éditez une carte vous aurez une grille devant vous avec des murs en bleu et le sol en gris.
La case en rose est le point d'apparition, vous pouvez le déplacer en cliquant sur "spawn".
Pour sauvegarder la carte que vous venez de faire, cliquez sur save, puis renseignez le nom de votre carte
sur la fenêtre qui vient de s'ouvrir et revalidez avec "save" sur cette même nouvelle fenêtre.

Dans tous les menus, "back" vous permet de revenir en arrière.
